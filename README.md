# 김현승

## 👋 소개
주어진 업무에 항상 '책임감'을 가지며, 주변과 '소통'을 게을리하지 않으며, 꾸준히 '성장'하고 자신을 돌보는 **개발자, 김현승**입니다!

## 🚀 경력

- **2022.02 ~ 현재:** 선도소프트(GIS 사업부, BI 사업부 / 주임)
  - 해양수산빅데이터플랫폼 개발
  - 폭염 상황관리 대시보드 개발
  - 새만금 해양환경·사회경제 통합정보시스템 개발
  - 해상풍력 상충분석 모바일 프로토타입 개발
  - 인도양 생명 열수공 DB GIS 시스템 개발
  - 디지털 활용 현안 해결 온라인 플랫폼 개발
  - 감염병 재난상황대응 GIS기반 의사결정지원 시스템 연계 개발
  - 데이터 구축 사업 인원관리
- **2021.09 ~ 2022.01:** 이노티브아이엔씨(개발 부서 / 인턴)
  - 제휴사 주간 통계 메일 발송 시스템 제작
  - 사내 도서관/도서 대여 시스템 제작
  - 회사 홈페이지 제작
  - 사내 서비스 유지보수
- **2019.11 : 제2회 ICT CoC(ICT complex of ceative developer) 해커톤** : 우수상
- **2019.11 : 제3회 인공지능 챗본톤(2019 AI WEEK KOREA)** : 장려상
