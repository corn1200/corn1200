# 김현승

## 👋 소개

- **2022.02 ~ 현재:** 선도소프트(GIS 사업부, BI 사업부 / 주임)
  - 해양수산빅데이터플랫폼 개발
  - 폭염 상황관리 대시보드 개발
  - 새만금 해양환경·사회경제 통합정보시스템 개발
  - 해상풍력 상충분석 모바일 프로토타입 개발
  - 인도양 생명 열수공 DB GIS 시스템 개발
  - 디지털 활용 현안 해결 온라인 플랫폼 개발
  - 감염병 재난상황대응 GIS기반 의사결정지원 시스템 연계 개발
  - 데이터 구축 사업 인원관리

  - 사내 도서관/도서 대여 시스템 제작
  - 회사 홈페이지 제작
  - 사내 서비스 유지보수
